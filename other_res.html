Learn Web Development - Free

While a coding bootcamp can take your skills to a professional level, it's important to take the first step by trying out one of numerous free, self-guided online web development courses that cover everything from basic programming introductions to development best practices. Below are some of the sites that provide free web development courses.

  • Mozilla Developer Network
  • HTML5 Rocks
  • A List Apart
  • Coding Dojo Algorithm Prep
  • The Odin Project
  • Free Code Camp
